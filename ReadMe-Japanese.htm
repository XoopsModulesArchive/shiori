このモジュールについて

はじめに「しおり」と読みます。ご存知、読みかけの本にはさむ「枝折り」のことです。このモジュールは、各ユーザーが個人のブックマーク(お気に入り)帳を所有でき、サイト内のあらゆるページをブックマークすることができるモジュールです。読みかけのページ、毎日チェックするスレッド、残しておきたい有用な記事などを見失わないよう、リンクを保存することが目的で開発されました。「ブックマーク」ブロックを表示することで、ユーザーがブックマーク帳にページを追加出来るようになります。

機能説明

  • ブックマーク帳を作る機能
  • ブックマーク登録の上限(容量)設定可能
  • ブックマークの期間人気ランキング閲覧機能

インストール方法

1) 圧縮ファイルを解凍してモジュールディレクトリにアップロード。

2) モジュール管理にてインストールしてください。

3) インストール完了画面に表示される「セットアップする」をクリックしてセットアップ行って下さい。

アンインストール方法

通常のモジュール同様にモジュール管理よりアンインストールを実行してください。

Tips - テーマに「このページをブックマークに追加」を表示 -

テーマファイル(theme.html)の好きなところに下記のコードを追加してください。 
<a href="<{$xoops_url}>/modules/shiori/bookmark.php?url=http://<{php}>echo $_SERVER['HTTP_HOST'].$_SERVER['REQUEST_URI'];<{/php}>">このページをブックマークに追加</a>
「このページをブックマークに追加」というテキストは画像に変更したりすると、見栄えが良くなるかも知れません。
