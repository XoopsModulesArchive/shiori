SHIORI

Sobre este módulo

Significa algo así como primera "marca". Lo mas parecido es el "marcador de páginas de un libro" que estás leyendo. Este módulo permite que cada usuario pueda poseer un libro con marcadores personales (favoritos) , y marcar cualquier página de un sitio. El propósito es guardar el enlace a una página que no has acabado de leer, que puedas continuar una secuencia cada día, o un informe útil antes de irte, etc, para que no se pierda. Cuando se mostra el bloque "Marcador", un usuario puede añadir una página al instante en su libro de marcadores.

Explicación funcional

  • Su función es la de un marcador de libros.
  • Es posible configurar la capacidad máxima de registros de marcadores
  • Clasificación de marcadores mas populares, durante un periodo de tiempo.

Método de instalación

1) Descomprime el archivo comprimido y bájalo al directorio de módulos (/xoops/module/). Puede asignarle el nombre de directorio de módulo que desee.

2) Por favor, instalarlo desde la administración de módulos. Y no vuelvas atrás 

3) Debajo del texto de instalación del módulo, hay un enlace para continuar con las instalación de "Opciones de configuración iniciales" y que se irán mostrando hasta la completa instalación.

Método de desinstalación

Por favor, realice la desinstalación del módulo como cualquier otro módulo habitual.
